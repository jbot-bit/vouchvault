# VouchVault — Handoff

VouchVault V3 has shipped (chunks 1–18 complete on `main`). The runtime is a plain Node Telegram bot: `src/server.ts` answers `/webhooks/telegram/action`, delegates to `src/telegramBot.ts`, and reads/writes Postgres via Drizzle. There is no Mastra, no Inngest, and no LLM in the hot path — do not reintroduce that stack. Operational follow-ups (Railway deploy, smoke tests, env vars) live in `DEPLOY.md`; legacy-replay tooling sits under `scripts/`. The canonical project guide for new contributors is `CLAUDE.md`, and the V3 spec is `docs/superpowers/specs/2026-04-25-vouchvault-redesign-design.md`.
